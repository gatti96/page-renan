Projeto Voar Agora

Este projeto é um site desenvolvido com HTML, CSS.
Voar Agora foi um dos primeiros sites que desenvolvi, apenas com um conhecimento inicial de HTML e CSS.
A princípio ele foi feito sua organização usando somente margin e sem nenhuma responsividade e após adquirir novos conhecimentos em CSS atualizei o projeto, agora utilizando Flex Box e Grid para melhor organização do site, assim como, também foi feita a responsividade.

Repositório: Link GitHub.
Deploy: Voar Agora.
